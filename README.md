# Verification Code
This is an in-progress repository for code related to the verification of climate models. It contains functions for evaluating the performance of climate models and downscaling models where the prediction can be compared to a target or truth (observations or target model).

## skill
Simple skill based functions such as correlation and rmse between sets of paired predictions and target.

## object_based
Object based assessments for sets of paired predictions and target e.g. for events where the climate variable is beyond a given threshold.
